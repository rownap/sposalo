# Sposalo

Sposalo is a portfolio-ready Next.js SaaS concept for Italian wedding planners.

It includes:
- A conversion-focused landing page at `/`
- A polished login experience at `/login`
- An operational dashboard at `/dashboard`
- Additional dashboard sections for projects, clients, contracts, and finances

The product intent is to replace fragmented workflows (spreadsheets, chat, and docs) with one workspace for quotes, timelines, contracts, and budget visibility.

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run start
```

## Project Routes

- `/` landing page
- `/login` login screen
- `/dashboard` main dashboard
- `/dashboard/projects`
- `/dashboard/clients`
- `/dashboard/contracts`
- `/dashboard/finances`
